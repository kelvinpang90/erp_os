# Malaysian Company & Person Name Patterns

> 生成 supplier / customer / contact person 时使用此模板，避免"John Doe"式假名。

## 公司名模板

### 1. 华商公司（约 60% 马来中小企业）

```
格式 A: {华人姓氏} {商业类别} Sdn Bhd
  Tan Chong Trading Sdn Bhd
  Lim Hong Enterprise Sdn Bhd
  Ng Brothers Holdings Sdn Bhd
  Chong Wei Industries Sdn Bhd

格式 B: {华人姓氏} & {华人姓氏} {类别}
  Lim & Tan Sdn Bhd
  Wong & Sons Trading
  Ooi & Associates Distributors

格式 C: 地域 + 商业
  Penang Heng Trading Sdn Bhd
  KL Central Supplies Sdn Bhd
  Johor Furniture Mfg Sdn Bhd

格式 D: 寓意 + 商业
  Fortune Link Sdn Bhd       # 好运
  Gold Harvest Enterprise    # 金丰收
  Jade Dragon Trading Sdn Bhd
```

华人姓氏池：Tan, Lim, Wong, Chong, Ooi, Ng, Chua, Goh, Lee, Cheong, Teh, Yeoh, Ang, Koh, Neoh

### 2. 马来商公司（约 30%）

```
格式 A: Syarikat {马来名} Sdn Bhd
  Syarikat Ahmad Ismail Sdn Bhd
  Syarikat Kilang Makanan Mesra
  Syarikat Perniagaan Maju Jaya

格式 B: {马来名词} {形容词} Sdn Bhd
  Maju Jaya Trading Sdn Bhd
  Sri Murni Enterprise Sdn Bhd
  Bintang Mewah Sdn Bhd
  Kilang Setia Sdn Bhd

格式 C: {专有词} Industries / Holdings
  Nusantara Industries Sdn Bhd
  Tenaga Holdings Sdn Bhd
```

马来关键词池：Maju, Jaya, Mewah, Setia, Murni, Harmoni, Bintang, Sri, Perkasa, Nusantara, Tenaga

### 3. 印度商公司（约 10%）

```
格式 A: {印度姓氏} Brothers / & Sons
  Raju Brothers Distributors Sdn Bhd
  Krishnan & Sons Trading
  Patel Brothers Enterprise

格式 B: {神名} Trading / Enterprise
  Ganesh Trading Sdn Bhd
  Kumaran Enterprise
```

印度姓氏池：Raju, Krishnan, Patel, Kumaran, Ramasamy, Subramaniam, Ravichandran, Muthu

### 4. 马来大型企业（少量，用于 B2B 大客户）

```
Tan Chong Motor Holdings Berhad   # 上市公司用 Berhad
Genting Berhad
IOI Corporation Berhad
Sime Darby Plantation
Maxis Broadband Sdn Bhd
```

## 人名模板

### 华人名字

```
{英文名} {姓} {中间字} {结尾字}
  Kelvin Tan Chong Wei
  Jessica Lim Hui Ling
  David Ng Kok Seng
  Sarah Ooi Mei Yee

纯中文名:
  陈志伟 / 林美玲 / 黄国华 / 吴丽萍
```

### 马来名字

```
{个人名} bin {父亲名}         # 男性
  Ahmad bin Ismail
  Mohammad bin Hassan
  Firdaus bin Azmi

{个人名} binti {父亲名}       # 女性
  Siti binti Abdullah
  Nurul Aini binti Rahman
  Farah Izzati binti Ahmad
```

### 印度名字

```
{个人名} a/l {父亲名}          # 男性（a/l = anak lelaki）
  Ravi a/l Krishnan
  Kumaran a/l Subramaniam

{个人名} a/p {父亲名}          # 女性（a/p = anak perempuan）
  Priya a/p Ramasamy
  Lakshmi a/p Muthu
```

## TIN（Tax Identification Number）格式

```
企业：C + 10 位数字
  C1234567890
  C9876543210

个人：12 位数字（NRIC 格式，YYMMDD-PB-XXXX 去掉破折号）
  800101015234  # 1980-01-01 出生，沙巴（15），男（奇数结尾）
  920415086721

e-Invoice 场景：
  B2B 买家：必填企业 TIN
  B2C 买家：可填个人 NRIC / TIN；无则用 "000000000000"
```
